## 주요 기능

1. **회원가입**
   - 개인/법인 회원 선택
   - 실시간 폼 검증
   - 중복 아이디/이메일 체크
   - 회원가입 성공 시 자동 로그인

2. **로그인**
   - 아이디/비밀번호 로그인
   - 에러 메시지 표시
   - 로그인 성공 시 홈으로 이동

3. **인증 상태 관리**
   - Zustand를 사용한 전역 상태 관리
   - localStorage를 통한 토큰 영속화
   - 페이지 새로고침 시 인증 상태 복원

4. **Protected Route**
   - 인증이 필요한 페이지 보호
   - 미인증 시 자동 리다이렉트

5. **회원 정보 관리**
   - 회원 정보 조회
   - 회원 정보 수정
   - 로그아웃 기능

## UI/UX 특징

- 로딩 상태 표시
- 사용자 친화적인 인터페이스
