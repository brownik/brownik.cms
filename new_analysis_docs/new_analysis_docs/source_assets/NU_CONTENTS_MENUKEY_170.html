오픈 API활용신청은 활용목적에 따라 두 종류로 구분됩니다.

  • 개발계정
    • 오픈 API 활용신청 시 첫 번째 활용단계입니다.
    • 제공기관에서 정의한 트래픽을 초과하여 활용할 수 없습니다.
  • 운영계정
    • 개발계정 다음 단계로 신청을 통해 진행할 수 있습니다.
    • 운영신청 시 개발된 어플리케이션에 대해 내용을 활용사례로 등록해야 합니다.

오픈 API는 활용신청 후에 두 가지 방식으로 승인이 이루어집니다.

  • 자동승인
    • 활용신청에 대해 담당자의 별도 확인 없이 신청과 동시에 활용할 수 있습니다.
  • 심의승인
    • 활용신청에 대해 담당자가 확인 후 '승인' 이후 부터 사용할 수 있습니다.
    • 신청 후 2~3일이 소요되고 결과는 회원가입 시 등록한 이메일로 발송합니다.

오픈 API를 활용하기 위해서는 반드시 '서비스키'를 발급 받아야 합니다.

  • 서비스키는 1인당 하나만 발급됩니다.
  • 서비스키를 재발급 받게 되면 이전에 발급한 키는 자동으로 폐기됩니다.
  • 서비스키 발급절차는 공공데이터포털(http://www.data.go.kr)에서 이루어지며, 하나의 서비스키로 '광산imap'과 공공데이터포털에서 제공하는 모든 오픈 API 서비스를 활용할 수 있습니다.

광산구 데이터 포털에서는 오픈 API활용을 제한할 수 있습니다.

  • 활용제한의 경우
    • 상업적 이용이 불가능한 API에 대해 상업적 활용한 경우
    • 트래픽 초과로 인해 시스템에 부화가 발생할 경우
    • 기타 등의 이유로 제한될 수 있습니다.
    • 마이페이지 > 오픈 API 에서 각각의 상세한 제한사유를 확인 할 수 있습니다.
  • 활용제한 종류
    • 오픈 API 내 상세기능 단위로 활용하지 못하도록 제한합니다.
    • 오픈 API 내 상세기능의 일일트래픽을 활용자가 신청한 트래픽 이하로 제한합니다.

개발계정 신청 절차

  • Step01
    • 광산imap 에서 API를 활용하기 위해서는 "공공데이터 포털"사이트에 회원가입이 되어있어야 합니다. 반드시 공공데이터포털(http://www.data.go.kr) 미리 회원가입을 해두시기 바랍니다.
  • Step02
    • 광산imap 사이트에서 '공공데이터' 메뉴의 '오픈 API' 메뉴를 클릭하면 이용가능한 API목록이 나옵니다. 목록에서 원하는 API를 선택한 후 상세화면으로 이동합니다.
  • Step03
    • 선택한 오픈 API의 상세화면에서 데이터포맷, End Point, 기술문서, 요청변수, 출력결과, 샘플데이터 확인, 에러메세지 등을 확인하고, 서비스를 이용하고자 하는 경우, [활용신청] 버튼을 클릭합니다.
  • Step04
    • 로그인을 합니다. [활용신청]버튼을 클릭하면 Step01에서 가입한 공공데이터 포털 계정으로 로그인을 해야 이용이 가능합니다.
  • Step05
    • API개발계정신청 양식에 맞게 입력합니다.
  • Step06
    • 신청 후 승인절차(자동승인 혹은 심의)가 이루어집니다. 자동승인인 경우 일정시간이 지나면(약 20~30분 후)에 [마이페이지 > API활용현황]에서 신청한 API정보를 확인할 수 있습니다.
  • Step07
    • 활용신청 승인이 완료되면, [마이페이지 > 개발계정]에서 서비스명을 클릭 후 상세보기 화면에서 인증키발급프로그램을 다운받아 서비스키를 발급받거나, 공공데이터포털(http://www.data.go.kr)에서 발급한 서비스키를 활용할 수 있습니다.
  • Step08
    • 테스트 용도로 오픈 API를 활용합니다.

운영계정 신청 절차 (개발계정 신청 후)

  • Step09
    • [마이페이지 > API 활용현황]에서 운영계정으로 전환하고자 하는 API의 [운영계정신청] 버튼을 눌러 운영계정으로 전환 신청을 할 수 있습니다.
  • Step10
    • 운영계정으로의 승인절차가 이루어집니다.
  • Step11
    • 운영계정으로 승인이 완료되면 실제 서비스를 운영하기 위한 목적으로 오픈 API 활용이 가능합니다.
