주민 참여로 안전한 광산을 만드는 참여와 소통으로 지역을 바꾸는 지도입니다.

  • 맘편한 광산은 지역주민들이 만드는 지도입니다. 주민들이 수집한 생활안전정보를 모바일 앱과 웹페이지를 통해 공유합니다.
  • 주민과 협력하여 지역문제를 해결하는 광산형 '정부 3.0' 정책입니다.

맘편한 광산은 주민들이 직접 생활안전정보를 생산하는 모바일 앱과 생산된 생활안전정보를 열람할 수 있는 웹페이지로 구성되어 있습니다.

  • 맘편한 광산 모바일 앱을 이용해 주민들이 직접 지역의 '두려운 곳, 보기안좋은 곳, 보행위험, 안전시설, 기타' 등의 생활안전 정보를 등록할 수 있습니다. 등록하신 정보(민원)는 광산구 홈페이지 구민의 소리와 연동되어 담당 직원이 처리한 결과를 주민에게 공개합니다.
