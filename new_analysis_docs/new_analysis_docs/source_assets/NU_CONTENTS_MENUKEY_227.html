마을e척척은 지역의 뿌리인 마을에서 지역의 문제를 해결하는 활동을 도와주는 플랫폼입니다.

  • 문제해결을
    위해 함께 만드는
    참여형 지도

    동네한바퀴

  • 마을의 문제를
    찾기 위한
    주민들의 소통 공간

    마을의제

  • 의제를 결정하고
    문제를 해결하기
    위해 투표

    마을총회

  • 결정된 마을
    실천의제를 논의하고
    진행과정을 기록

    마을활동

  • 마을의 변화와
    문제해결 활동의
    결과를 공유

    마을영상관

  • 마을이 가진
    자원을 확인해보는
    자원지도

    마을자원지도

마을e척척은 행정안전부가 주최하고, 한국정보화 진흥원이 주관하는 ‘2019년 디지털 사회혁신 프로젝트'의 지원을 받아 개발하였습니다.
