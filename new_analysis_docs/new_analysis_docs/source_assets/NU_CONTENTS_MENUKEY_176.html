인구 현황(2017)

2017년 기준, 광산구의 인구는 404,131명으로, 2016년 대비 0.3%(1,082명) 증가함

그림 1> 2005년 대비 2017년 광산구 인구변화, 데이터 출처: 인구주택총조사(통계청, 2005), 주민등록인구 현황(광산구 주민자치과, 2017)

2017년 기준 광산구의 인구변화를 2005년 인구 현황 비교하여 인구가 증가한 지역과 감소한 지역을 분석함(2017년 주민등록인구 – 2005년 인구주택총조사). 데이터 집계 형태가 다른 인구주택총조사와 주민등록인구 데이터를 비교하기 위해 인구주택총조사의 집계구 구역을 기준으로 데이터를 변형시킴. 광산구에서 인구가 급격하게 증가한 지역은 수완·신창·신가 지역과 하남2지구, 선운지구로 신규 택지개발사업으로 인해 아파트 대단지가 유치된 지역임

그림 2> 광산구 인구 변화(2000~2017), 데이터 출처: 주민등록인구 통계(통계청, 2000~2017)

2000년부터 2017년 사이 광산구의 인구변화와 전년 대비 인구 증가율을 나타냄
2017년 기준(외국인 제외), 광산구의 인구는 404,131명으로 2016년 대비 0.3%(1,082명) 증가함
광산구는 2008년 수완지구 택지개발사업이 마무리되면서 수완·신가·신창동을 중심으로 인구가 급격하게 증가함. 수완동은 현재(2017년 말 기준) 인구 80,252명으로 성장했으며, 광산구의 인구증가를 견인하였음
광산구의 인구증가율은 수완지구 택지개발사업이 2008년에 마무리된 후 2009년에 6.9%(22,276명), 2010년 5.9%(20,303명)로 급격하게 증가했다가 2017년 0.3%(1,082명)로 감소한 상태로 광산구의 인구변동이 안정화되고 있음

택지개발사업이 광산구 인구 증가를 견인함

그림 3> 광산구 공동주택 현황 , 데이터 출처: 공동주택 현황(광산구 주택과, 2017)

2017년 말 기준 광산구 공동주택을 공급세대 기준으로 나타냄. 원의 크기가 클수록 공급세대가 많은 공동주택임. 공동주택을 광산구의 인구가 급격하게 증가하는 2009년부터 공급되기 시작한 사용 검사년도 2008년을 기준으로 구분함. 공동주택 현황은 광산구 주택과에서 관리하는 아파트 총 255 단지를 대상으로 함(2017년 말 기준)

그림 4> 광산구 공동주택 현황, 데이터 출처: 공동주택 현황(광산구 주택과, 2017)

- 세로 막대그래프는 행정동별 공동주택 현황을 나타내며, 바 그래프는 행정동별 공동주택의 총 공급세대를 나타냄
광산구의 인구는 새로운 택지개발사업을 통해 증가해왔으며, 2000년 이후 수완지구 택지개발사업, 하남2지구 택지개발사업, 선운지구 택지개발사업을 통해 대단위 아파트 단지가 형성되고 급격하게 인구가 증가해왔음(그림1의 인구가 증가한 지역(빨간색)과 그림3의 ‘사용 검사년도 2008년’부터 공급된 공동주택(빨간색) 분포가 일치하며, 이는 택지개발사업을 통해 인구가 증가했음을 확인할 수 있음)
현재(2017년 말 기준) 인구 80,252명으로 인구가 급격하게 증가한 수완동은 택지개발사업이 완료된 2008년 이후 48단지 23,841 세대의 아파트가 공급됐으며, 어룡동은 택지개발사업 후 현재 9단지 4,947 세대가 공급돼 2010년 20,883명에서 2017년 33,191명으로 증가함
행정동별 아파트 현황은 총 255단지 가운데, 수완동이 48단지 23,841세대로 가장 많으며, 다음으로 어룡동이 36단지 11,234세대, 첨단1동이 24단지 7,112세대, 첨단2동이 24단지 11,924세대 순으로 나타남
