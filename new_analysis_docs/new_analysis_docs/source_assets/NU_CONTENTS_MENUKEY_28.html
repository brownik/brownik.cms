성북동 유래

혜화문과 숙정문 사이의 서울 성곽이 부채꼴 모양으로 감싼 성북동은 조선시대 도성 수비를 담당했던 어영청의 북둔(北屯)이 영조 41년(1765)에 설치된 연유로 동명이 붙여졌다.

동성 4소문의 하나인 혜화문을 나서서 왼쪽 일대의 계곡마을인 성북동은 예부터 맑은 시냇물이 흐르고 수석(水石)이 어울린 산자수명(山紫水明)한 마을로 복숭아, 앵두나무가 많아서 사람들의 사랑을 많이 받아온 곳이다.

1948.08.13 동대문구 성북동을 성북구 성북동으로 분리신설 (대통령령제159호)
1949.08.15 성북구 성북1,2가를 구분
1955.04.19 동제실시로 성북동 1가지역을 성북1동 행정동으로 관할
2007.12.30 성북1동,성북2동, 동소문1가,동소문4가를 성북동으로 통합하여 관할
